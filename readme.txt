Power Set Calculator v 1.0
Created for Revolution UC 2019 Hackathon Season
(c) David Gonzalez

Directions:
  - enter elements into text box, separate them using commas.
  - press "Find Subsets"

Bugs:
  - Entering the elements incorrectly may lead to incorrect subsets
  - Leaving the box empty returns two (2) empty sets
  - Passing in a set as an element is breaks program.
    * Fix *
      - Enter sets, with elements in the sets separated with spaces or fullstops.
        ex: enter {1,2,3} as {1 2 3} or {1.2.3}
